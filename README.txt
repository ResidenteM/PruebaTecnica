# Instrucciones para correr la página web

## Requisitos previos
1. Node.js instalado. Descarga aquí: https://nodejs.org/
2. Navegador moderno (Chrome, Edge, Firefox).
3. Conexión a internet para cargar Google Maps.

## Pasos para configurar y ejecutar el proyecto
1. Descomprimir todos los archivos en una carpeta.
2. Abrir una terminal o línea de comandos y navegar hasta la carpeta: cd C:\ruta\de\la\carpeta
3. Instalar las dependencias necesarias con:
npm install
4. Iniciar el servidor con:
node server.js
5. Abrir el navegador y acceder a la página web:
http://localhost:3000
